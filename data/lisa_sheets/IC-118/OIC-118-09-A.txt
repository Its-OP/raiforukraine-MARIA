Screening test for memory impairment :

- Mini Mental State (MMS): assessment of cognitive functions, particularly memory, used mainly in cases of suspected dementia.


Dubois 5-word test:

o Present a pre-established list of 5 words aloud,

o Request them again by category,

o Immediate free recall, then code if necessary,

o Interference test (clock test),

o Free recall, then coded if necessary.
